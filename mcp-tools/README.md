# Model Context Protocol (MCP) Tools

This directory contains AI-driven tools that integrate with the AetherOS platform.
The MCP foundation allows AetherOS to contextually interact with LLMs for:
- Automated security vulnerability detection.
- Intelligent deployment optimization.
- Contextual logging analysis.

To add a new tool, create a new subdirectory and implement the MCP server using the SDK.
Check https://modelcontextprotocol.io for more information.
